## Milestone Summary

**Key Decisions:**
- D020 — Refusal detection is pattern-based regex, not Haiku-classified (cheap, deterministic, good enough for bounded problem)
- D021 — Language detection uses `franc` with minimum-length threshold (<4 words / <15 chars inherits prior)
- D022 — Constitutional preamble is a floor, not a ceiling (additive to existing mode prompts)
- D023 — Live integration tests assert absence of bad behavior, 3-of-3 passes against real Sonnet
