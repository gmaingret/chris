    **Step 3 — Create `src/proactive/triggers/deadline.ts` (per D-01, D-02, D-03, D-08, D-09):**
    Follow the `createSilenceTrigger()` factory pattern from `silence.ts`. The factory function `createDeadlineTrigger()` returns a `TriggerDetector` with a `detect()` method.

    Export `STALE_CONTEXT_THRESHOLD_MS = 48 * 60 * 60 * 1000` as a named constant.

    `detect()` logic:
    1. Define `notTriggered(context)` helper returning `{ triggered: false, triggerType: 'decision-deadline', priority: 2, context }`.
    2. Query: `db.select().from(decisions).where(and(eq(decisions.status, 'open'), lte(decisions.resolveBy, now))).orderBy(asc(decisions.resolveBy)).limit(1)`. Use `const now = new Date()`.
    3. If no rows, return `notTriggered('No due decisions')`.
    4. Try `await transitionDecision(candidate.id, 'open', 'due', { actor: 'sweep' })`.
    5. Catch `OptimisticConcurrencyError`: re-run the same SELECT query. If no rows, return `notTriggered('No due decisions after retry')`. Otherwise call `transitionDecision` on the new candidate (no second retry on second failure — let it throw).
    6. Catch `InvalidTransitionError`: return `notTriggered('Decision already transitioned')`.
    7. Re-throw any other error.
    8. Build context string per D-08:
       - `const staleness = now.getTime() - candidate.resolveBy.getTime();`
       - If `staleness > STALE_CONTEXT_THRESHOLD_MS`: `"On ${candidate.resolveBy.toISOString().slice(0, 10)} you predicted: '${candidate.prediction}'. It's now ${Math.round(staleness / 86400000)} days past your deadline. Your falsification criterion was: '${candidate.falsificationCriterion}'."`
       - Else: `"Your deadline just passed for a prediction you made: '${candidate.prediction}'. Your falsification criterion was: '${candidate.falsificationCriterion}'."`
    9. Return `{ triggered: true, triggerType: 'decision-deadline', priority: 2, context, evidence: ['Decision ID: ...', 'Resolve by: ...', 'Staleness: ...h'] }`.

    Imports needed:
    ```typescript
    import { and, eq, lte, asc } from 'drizzle-orm';
    import { db } from '../../db/connection.js';
    import { decisions } from '../../db/schema.js';
    import { transitionDecision } from '../../decisions/lifecycle.js';
    import { OptimisticConcurrencyError, InvalidTransitionError } from '../../decisions/errors.js';
    import type { TriggerResult, TriggerDetector } from './types.js';
    ```
