1. Both triggers fire (both return triggered=true)
2. Sweep selects exactly one winner (the highest-priority = lowest number)
3. One message is sent (not two — single pipeline)
4. The non-winning trigger is not starved (it fired, it was considered, it just lost priority)

const VAGUE_TIMEOUT_MS = 15000;
